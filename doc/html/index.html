  • Visit the glean project page, which offers a central point of access to the main source code repository, bug database, mailing lists, etc.

  • Subscribe to the glean-announce mailing list, for announcements of new releases and other notices of interest to the glean user community.
  • Subscribe to the glean-dev mailing list, for discussions between glean developers.
